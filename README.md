# Weather Underground

Weather Underground operates a personal weather station network of 250,000+ stations worldwide, providing hyperlocal weather data via a REST API. The PWS API (hosted at api.weather.com) offers authenticated access to current conditions, historical observations, 5-day forecasts, and location lookup for registered station contributors. API keys are free for PWS owners who upload data to the network.

- **Website:** https://www.wunderground.com
- **API Base URL:** https://api.weather.com
- **API Keys:** https://www.wunderground.com/member/api-keys
- **Blog:** https://www.wunderground.com/blog
- **X:** https://x.com/wunderground
- **LinkedIn:** https://www.linkedin.com/company/wunderground-llc

## APIs

- **Weather Underground PWS API** — Current conditions, historical data, forecasts, and location lookup for Personal Weather Station contributors.
